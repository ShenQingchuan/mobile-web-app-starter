# Mobile Web App starter

## Tech Stack

- Vue.js 3.5+
- Vite PWA
- UnoCSS
- Varlet UI
- Eruda

## How to use

```bash
# Copy the .env.example file to .env.local
cp .env.example .env.local
rm .env.example

# Install dependencies
pnpm install

# Run the development server
pnpm dev
```
